# LSB Image Steganography

## Project Overview
This project implements LSB (Least Significant Bit) Image Steganography to hide secret data inside digital images.

The technique hides confidential messages inside the least significant bits of image pixels without significantly affecting the image quality.

## Features
- Hide secret message inside image
- Extract hidden message from stego image
- Maintains original image quality
- Secure communication method

## Technologies Used
- C Programming
- Linux
- Image Processing Concepts

## How It Works
1. Select a cover image.
2. Encode the secret message using LSB technique.
3. Generate stego image.
4. Decode the image to retrieve the hidden message.

## Applications
- Secure communication
- Digital watermarking
- Data protection
